Project Overview

About This Project

A deep learning system that classifies natural disasters from images using a Convolutional Neural Network trained on real disaster imagery — built with TensorFlow, Keras, and Flask.

The Problem

Natural disasters — cyclones, earthquakes, floods, and wildfires — cause catastrophic damage to ecosystems, infrastructure, and human lives every year. Traditional detection methods rely on manual assessment, which is time-consuming and often delayed. Early and accurate classification of disaster types from visual data can significantly improve emergency response times.

Many deep learning approaches struggle with complex and imbalanced disaster image structures. This project addresses that challenge by developing a robust CNN model capable of classifying disaster intensity from images in real time.

⏱️

Slow Detection

Manual identification of disaster types takes hours, delaying critical response efforts.

📊

Imbalanced Data

Disaster image datasets are often imbalanced, making classification harder for ML models.

🌐

Real-time Need

Emergency services need instant classification to prioritize and allocate resources effectively.

🔍

Visual Complexity

Disaster images vary widely in angle, lighting, and scale making pattern recognition difficult.

Model Architecture

Disaster Classes

The model detects and classifies images into 4 natural disaster categories, each with distinct visual patterns the CNN learns to identify.

🌀
Cyclone
Rotating storm systems characterized by spiral cloud patterns and strong winds, typically forming over warm ocean waters.
⚠️ Evacuate coastal areas and seek sturdy inland shelter immediately.
🏔️
Earthquake
Sudden seismic events causing ground shaking and structural collapse, often visible through damaged buildings and debris.
⚠️ Drop, cover, and hold on. Stay away from windows and heavy objects.
🌊
Flood
Overflow of water submerging normally dry land, identifiable by submerged structures and muddy water patterns in imagery.
⚠️ Move to higher ground immediately. Do not walk through floodwaters.
🔥
Wildfire
Uncontrolled fires spreading through vegetation, identified by visible flames, dense smoke plumes, and charred terrain.
⚠️ Evacuate immediately. Close all windows and doors before leaving.
Project Flow
1

Data Collection & Augmentation

939 images across 4 disaster classes collected. ImageDataGenerator applied rescaling, shear, zoom, and horizontal flip to improve model generalization.

2

Model Building

Sequential CNN built with Keras — two Conv2D + MaxPooling blocks, a Flatten layer, and Dense output with Softmax activation for multi-class classification.

3

Training

Model trained for 20 epochs with Adam optimizer and categorical crossentropy loss. Achieved 91%+ training accuracy and ~78.7% validation accuracy.

4

Model Saved

Trained model saved as disaster.h5 for reuse without retraining. Weights and architecture fully preserved.

5

Flask Web Application

Flask app built to serve predictions via a web interface. Users upload an image, the model predicts the disaster type, and results are displayed with confidence scores and safety tips.
